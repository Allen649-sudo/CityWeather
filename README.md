 WeatherApp – Android-приложение для отслеживания погоды в любом городе мира.
Пользователь может искать города, просматривать детальную информацию о погоде (температура, влажность, ветер) и добавлять любимый город в список избранных.
Есть список популярных городов, который уже присутствует при запуске приложения, а также возможность установить город по умолчанию, чтобы прогноз открывался сразу для него.

Стек технологий:  Kotlin, Jetpack Compose, MVVM + ViewModel + StateFlow, Retrofit, Room, Coroutines, пагинация, Pull-to-refresh
